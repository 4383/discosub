Discosub

Simple and faster subdomain discover

Download as .zip Download as .tar.gz View on GitHub

Welcome to Discosub official website.

Discosub is a fingerprint tools that provide subdomain discover scanning on a root domain.

Usages

via pipy:

$ pip install discosub
$ discosub run <specified root domain>

via github:

$ git clone https://github.com/4383/discosub
$ cd discosub
$ python setup.py install
$ discosub run <specified root domain>

Features

  • Root domain subdomain discover scanning.
